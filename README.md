# El Viaje del Descubrimiento

Este proyecto es una presentación interactiva en formato cómic, creada con HTML y Tailwind CSS, que narra el proceso de autodescubrimiento personal a través de 7 etapas visuales.

## Tecnologías Utilizadas

- **HTML5**
- **Tailwind CSS** (CDN)
- **Lucide Icons** (CDN)
