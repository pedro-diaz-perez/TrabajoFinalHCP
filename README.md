# El Viaje del Descubrimiento

Este proyecto es una presentación interactiva en formato cómic, creada con HTML y Tailwind CSS, que narra el proceso de autodescubrimiento personal a través de 7 etapas visuales.

## Estructura del Proyecto

- **presentacion final/text.html**: Archivo principal del cómic interactivo.
- **presentacion final/**: Carpeta donde se encuentran las imágenes y recursos utilizados en la presentación.
- **docs/**: Documentación y recursos adicionales (HTML, CSS, imágenes).
- **peuwb/**: Materiales y temas complementarios en HTML.
- **tets hhcp/**: Tests y recursos de apoyo.

## Tecnologías Utilizadas

- **HTML5**
- **Tailwind CSS** (CDN)
- **Lucide Icons** (CDN)

## Cómo visualizar la presentación

1. Abre el archivo `presentacion final/text.html` en tu navegador web.
2. Navega por las diferentes etapas del cómic para seguir la historia.

## Personalización

- Puedes cambiar las imágenes de cada etapa editando las rutas en el archivo `text.html`.
- Los estilos pueden modificarse directamente en el archivo o usando las utilidades de Tailwind CSS.

## Páginas principales 📄

- **Inicio** (`index.html`): Una sección principal que contiene información general sobre el sitio web.
- **Sobre** (`about.html`): Una sección acerca de mí que incluirá una breve descripción sobre mí inspirada en las unidades 2 y 3 (Módulo 1), un currículum vitae, dos imágenes y algunos enlaces a cuentas de redes sociales.
- **Contacto** (`contact.html`): Una sección de contacto que incluirá un formulario fácil de usar para ponerse en contacto conmigo. Este formulario está diseñado sin ningún elemento de script que envíe o guarde el contenido ingresado.
- **Grado** (`degree.html`): La sección de grado proporcionará una descripción detallada de mi grado, incluido su propósito, estructura y una lista completa de las materias cubiertas en el programa, cada una de ellas descrita brevemente. Entre estas materias, los "Fundamentos de la Ingeniería Informática" tendrán una página dedicada (`fi.html`) que ofrece una exploración en profundidad de su contenido, objetivos y resultados de aprendizaje.
- **Fundamentos (FI)** (`fi.html`): Una exploración en profundidad del contenido, objetivos y resultados de aprendizaje de la asignatura.
- **Red** (`net.html`): Una sección de red que proporcionará información (nombre y enlace) sobre las páginas de los compañeros, estableciendo enlaces a sus respectivos sitios web.
- **Tema** (`topic.html`): Una sección temática que ofrecerá información completa sobre un tema elegido, ya sea relacionado con el trabajo práctico o con un tema de interés personal.
